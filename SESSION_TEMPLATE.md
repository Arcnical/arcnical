## 💡 PRO TIPS FOR SESSION TEMPLATES

✅ **DO:**
- Be specific about WHAT (not vague like "work on code")
- Include inputs you're providing (req doc, files, screenshots)
- State expected output format (code? doc? checklist?)
- List blockers upfront

❌ **DON'T:**
- Try to do more than 3 tasks per session (context dilution)
- Skip the "Quick Context Recap" (I'll read the tracker, but recap helps you think)
- Leave "Success criteria" vague ("it works" vs "can import module + run tests")
